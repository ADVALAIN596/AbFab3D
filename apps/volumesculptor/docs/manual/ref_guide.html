Data Sources

Box

Box(sx,sy,sz)

Create a box centered around 0,0,0 with face sizes of sx,sy,sz.

Box(x,y,z,sx,sy,sz)

Create a box centered around x,y,z with face sizes of sx,sy,sz.

Cylinder

Cylinder(v0,v1,r)

Create a cylinder from position v0 to v1. Cylinder is of radius r.

DataSourceGrid

DataSourceGrid(grid)

Use a Grid as a datasource. Do not use a Grid as a datasource and a destination grid at the same time.

ImageBitMap

ImageBitMap(imagePath,sx,sy,sz)

Take an image as a source. The image will be the sx,sy,sz size in the voxel grid. Can take png, jpeg either as black and white or greyscale images. Greyscale images will act as heightmaps where 0 to no image and maxValue is full sized. Image placement can be specified via the setImagePlace function. An image can be placed in the TOP, BOTTOM or BOTH positions. In the TOP position the object grows upward from the starting point. In the BOTTOM it grows downward. The BOTH mode has it grow in both directions to create a mirror image. Follow this link for an exampleImage Example

Sphere

Sphere(c, r)

Create a sphere centered around Vector3d c with radius r.

Sphere(r)

Create a sphere centered around 0,0,0 with radius r.

Sphere(x,y,z,r)

Create a sphere centered around x,y,z with radius r.

Intersection

Intersection()

Intersection of arbitrary array of objects. Creates intersection with empty array of objects.

Intersection(ds1, ds2)

Intersects two data object. The result is any voxels which are shared between objects.

add(DataSource)

Add data source to the array of objects to intersect.

Ring

Ring(innerRadius, thickness, ymin, ymax)

Create a Ring in the XZ plane.

Ring(innerRadius, thickness, width)

Create a Ring in the XZ plane.

Subtraction

Subtraction(ds1,ds2)

Subtracts ds2 from ds1.

Torus

Torus(R, r)

Creates a torus. R is the outer radius, r is the inner radius. this.R = R; this.r = r; }

Union

Union(ds1, ds2)

The result is any voxel in either data source.

VolumePatterns

Volume Patterns fill the complete space with a repeating pattern.

Balls

Balls(period, radius)
A space filled with Spheres. Each sphere is period distance apart and radius sized.

CubicGrid

CubicGrid(period, radius)
A space filled with Cubes. Each cube is period distance apart and radius sized. See this reference for more info on Gyroids: en.wikipedia.org/wiki/Gyroid

Gyroid

Gyroid(period, thickness)
A space filled with a Gyroid function. Each gyroid is period distance apart and radius sized.

GyroidGradient

Gyroid(period, thickness,pos,str)
A space filled with a Gyroid function influenced by a gradient. Each gyroid is period distance apart and radius sized. Vector3d pos is the gradient center and str is the strength.

Transformations

CompositeTransform

Combines several transformations together. Use the add method to chain together several transformations.

Rotation

Rotation(axis, angle)

Rotate around a Vector3d axis for angle radians

Rotation(axis, angle, center)

Rotate around a Vector3d axis for angle radians with a specified Vector3d center

Scale

Scale(s)

Set the uniform scale s in all the x,y,z axises

Scale(sx,sy,sz)

Set the individual scale per axis.

Translation

Translation(tx,ty,tz)

Translate a source by x,y,z

Javascript Functions

createGrid

Create a voxel grid

Grid createGrid(sx,sy,sz,voxelSize)

load

Description: Load a 3D mesh into a grid.

Grid load(filename)

Returns a grid from the 3D mesh. Uses the default voxelSize of 0.1mm

Grid load(filename,voxelSize)

Returns a grid from the 3D mesh. Uses the specified voxelSize.

load(filename,Grid)

Load a 3D mesh into the specified grid. Uses the Grid's voxel size.

Supported formats are STL(ASCII and BINARY).
